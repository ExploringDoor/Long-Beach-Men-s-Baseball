League Alert Banner
Post emergency notices to the entire site instantly
❓

What does it do?

When you post an alert, a big red popup appears on the website for every person who opens it. They have to click OK to close it before they can use the site. It's impossible to miss — perfect for urgent news like rainouts.

✍️

How to post an alert

1. Type your message in the big text box
2. Click the red "POST ALERT TO SITE" button
3. Done — it goes live on the site immediately for everyone

🗑️

How to remove an alert

Clear the text box completely (delete everything you typed) and click "POST ALERT TO SITE" again with nothing in the box. The alert disappears from the site.

💡 When to use this: Rainouts · Game cancellations · Field changes · Time changes · Any urgent league-wide notice that everyone needs to see the moment they open the site
✅ Example messages:
"⚠️ RAINOUT — All Saturday April 19 games are CANCELLED due to rain. Makeup dates TBD."

"📍 FIELD CHANGE — April 26 games moved to Blair Field. Same times."

Enter Box Score
Enter results for any team — you see all games, not just one team
🔑

Admin vs. Captain — the key difference

When a captain logs in, they only see their own team's games. When you log in as admin, you see every game for every team in the season — all games from all matchups.

📋

Three tabs at the top

· FROM SCHEDULE — pick any game from this season and enter its box score
· EDIT SAVED GAME — find a game already submitted and fix a mistake
· CUSTOM GAME — enter a game that wasn't on the schedule (tournaments, makeups, etc.)

📌 The box score form is the same for admin as it is for captains — scores, linescore, batting lineups, pitching, recap. The only difference is you have access to all teams. See the Manager Guide for detailed instructions on filling out the form.
📁
Manage Games
Edit or delete any game that has already been submitted
🔍

What you'll see

A list of every submitted game — team names, date, and final score. Each game has an Edit button and a Delete button.

✏️

Editing a game

Click Edit next to any game. The full box score form opens pre-filled with all the existing data. Change whatever needs fixing, scroll down, and click "Update Box Score." Changes are live instantly.

🗑️

Deleting a game

Click Delete next to a game to permanently remove it from the database. This also deletes all batting and pitching stats tied to that game. Only delete if the game was entered by mistake or is a duplicate.

⚠️ Deleting is permanent. There is no undo. All stats for that game are erased from player records and leaderboards. If you just need to fix a score or stat, use Edit — not Delete.
📧
Send Weekly Email
Copy this week's results to paste into your email to the league
❓

What does it do?

This tool generates a formatted summary of the most recent week's scores and results — ready to copy and paste into an email. It pulls the latest game results automatically.

📋

How to use it

1. Click "Send Weekly Email" from the dashboard
2. The results summary appears on screen
3. Click the copy button to copy it all to your clipboard
4. Open your email, paste it in, and send to the league

💡 Note: This only works once box scores have been entered for that week. If you click it before any games are entered, it will say "No games recorded yet."
📅

Edit a game

Click the "✏️ Edit" button next to any game to change its date, time, field location, or teams. This updates the public schedule and the team calendars. Use this when a game gets rescheduled or moved to a different field.

➕

Add a game

Click the blue "+ Add Game" button in the top right to add a new game to the schedule — useful for makeup games or games that were added mid-season.

🔄

Reset button

The "Reset" button is for wiping the entire schedule and starting fresh. Do not click this unless you intend to completely rebuild the schedule. It cannot be undone.

⚠️ Do NOT click Reset unless you are 100% sure you want to erase the entire season schedule. This is a nuclear option — use Edit or Add Game instead.
